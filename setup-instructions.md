## Requirements

- Web browser
- Text editor (VS Code / Notepad++)
- VirusTotal account (optional)
- AbuseIPDB access

## Steps

1. Download phishing email sample (.eml)
2. Extract headers and identify sender details
3. Perform IOC extraction
4. Run IP reputation checks using VirusTotal and AbuseIPDB
5. Perform WHOIS lookup on relay domain
6. Classify incident using MITRE ATT&CK
7. Document findings in incident report

## Output Files

- header-analysis.txt
- ioc.txt
- threat-intel.txt
- incident-classification.txt
- incident-report.md
- screenshots folder

This project simulates a SOC Level-1 phishing investigation workflow.
